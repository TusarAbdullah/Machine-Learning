## Random Forest কী?

Random Forest অনেকগুলো Decision Tree একসাথে ব্যবহার করে সিদ্ধান্ত নেয়।
একটা গাছ না, বরং পুরো একটা বন (forest)

## Regression (যেমন: house price prediction)

## কিভাবে কাজ করে?

Dataset থেকে randomভাবে অনেকগুলো sample নেয়

প্রতিটা sample দিয়ে আলাদা আলাদা Decision Tree বানায়

সব Tree আলাদা আলাদা prediction দেয়

শেষে

Classification → majority vote

Regression → average নেয়

## কেন Random Forest ভালো?

Overfitting কম করে

Single decision tree থেকে বেশি accurate

Noise থাকলেও ভালো কাজ করে

Feature importance বের করা যায়
